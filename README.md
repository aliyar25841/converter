# Image Tools - Convertisseur & Éditeur d'Images

Une application web légère et rapide pour convertir, redimensionner et éditer vos images directement dans le navigateur. Absolument aucune donnée n'est envoyée sur un serveur (traitement 100% côté client).

## Fonctionnalités

### 1. Conversion de Format

- Convertissez vos images en **WebP**, **JPEG**, **PNG**, ou **AVIF**.
- Traitement par lots : convertissez des dizaines d'images en une seule fois.
- Téléchargement individuel ou groupé (archive ZIP).

### 2. Redimensionnement

- Redimensionnez par pourcentage (échelle) ou dimensions fixes (Largeur/Hauteur).
- Maintien du ratio d'aspect automatique.

### 3. Édition Rapide

- **Rotation** : Pivotez vos images par pas de 90°.
- **Miroir** : Inversez vos images horizontalement ou verticalement.
- Recadrage automatique (Carré, 16:9, 4:5, etc.).

## Points Forts

- **Confidentialité Totale** : Vos photos ne quittent jamais votre ordinateur.
- **Performance** : Utilise l'accélération matérielle du navigateur.
- **Interface Moderne** : Design soigné avec support du **Mode Sombre**.
- **Glisser-Déposer** : Interface intuitive pour l'ajout de fichiers.

## Comment l'utiliser

Il s'agit d'une application web statique. Aucune installation complexe n'est requise.

1. Clonez ce dépôt.
2. Ouvrez le fichier `index.html` dans n'importe quel navigateur moderne (Chrome, Firefox, Safari, Edge).
3. C'est tout !

## Technologies

- HTML5 / CSS3
- JavaScript (ES6+)
- [Tailwind CSS](https://tailwindcss.com/) (via CDN)
- [JSZip](https://stuk.github.io/jszip/) (pour les archives)
